## 项目栏目维护说明

- 按语言：Python / JavaScript / TypeScript

- 简介
- 适用场景
- 技术栈
- 难度
- 仓库链接

## 资讯栏目维护说明

### 原则

- 优先写精选解读，不做简单新闻搬运

- 更新频率可以低，但内容必须长期可读
