# Agent 面试题

Agent 题目通常比纯 LLM 问题更偏系统设计，重点在执行链路、状态管理、工具使用和可控性。

## 这部分常考什么

- Agent 和普通聊天机器人有什么区别
- 规划、记忆、工具调用分别承担什么职责
- 长记忆和短记忆如何协同
- ReAct、Plan-and-Execute 有什么差异
- 多 Agent 协同的收益、成本和风险是什么

## 回答这类题的关键

- 不要只讲概念，要能讲出执行流程
- 不要只说“会调用工具”，要说明如何做权限、失败重试和兜底
- 不要只谈理想架构，要说明什么场景下不该上多 Agent
