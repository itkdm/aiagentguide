# Prompt 面试题

Prompt 相关题目看起来门槛不高，但很适合区分候选人是否真的做过调试、评测和线上优化。

## 这部分常考什么

- Prompt 工程和微调分别适合什么场景
- 如何让模型输出更稳定
- 如何设计结构化输出
- 如何减少提示词漂移
- 系统提示词、开发者提示词和用户提示词如何分工

## 复习建议

- 不要只说“多试几版提示词”，要说明你如何验证效果
- 最好准备一个结构化输出或工具调用的真实案例
- 回答调优问题时，尽量体现约束、评测和迭代闭环
