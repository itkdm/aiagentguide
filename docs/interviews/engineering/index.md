# 工程化面试题

这部分最容易拉开差距，因为它能区分“做过 Demo”和“做过线上系统”的候选人。

## 这部分常考什么

- 怎么做离线评测和在线评测
- 如何监控模型效果退化
- 如何做 prompt、模型、检索策略的 A/B 实验
- 成本、延迟、成功率如何平衡
- 出现幻觉或工具调用失败时如何兜底

## 面试回答建议

- 尽量把指标讲具体，比如成功率、P95 延迟、单次成本
- 只说“做过监控”不够，最好说明监控对象、告警条件和处置流程
- 如果做过线上系统，优先讲实际故障、优化和复盘经验
