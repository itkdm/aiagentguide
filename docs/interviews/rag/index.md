# RAG 面试题

RAG 是 AI Agent 面试里最常见的模块之一，问题通常集中在流程理解、召回效果、评测方法和性能优化。

## 这部分常考什么

- 什么是 RAG，为什么不是所有问题都靠微调解决
- chunk 应该怎么切，切分策略会带来哪些影响
- 召回、重排、生成分别承担什么职责
- 如何评测召回效果和回答效果
- 如何处理知识过期、幻觉、延迟和成本问题

## 复习建议

如果你准备的是校招或转岗面试，先把流程和基本指标讲清楚。如果你准备的是中高级岗位，重点要放在评测体系、链路优化和项目取舍上。
