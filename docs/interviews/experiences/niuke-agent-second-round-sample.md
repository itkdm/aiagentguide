# AI Agent 二面题目

前半部分偏 RAG，后半部分逐步转向 Agent 和工程化，能够比较直接地看出一轮技术面试里的追问方式。

## 基本信息

- 方向：AI Agent
- 轮次：二面
- 面试时间：2026 年 3 月

## 题目清单

1. 做 RAG 项目时，通常如何评测效果，关注哪些维度和指标？
2. 项目里的数据集包含哪些内容，来源和格式是什么？
3. 如果要优化 RAG 的相关度和回答效果，有哪些系统化思路？
4. 一千条数据的清洗和处理流程应该怎么设计？
5. RAG 性能怎么提升，工程层和算法层分别怎么做？
6. 项目里的上下文是怎么处理的，超长上下文怎么优化？
7. Agent 的长记忆和短记忆如何协同，衔接逻辑是什么？
8. 你做的 Agent 如何进一步提升智能性？

## 这轮面试重点在考什么

- 前 1、3、5、6 题主要考 RAG 评测与优化能力
- 第 2、4 题更偏数据治理和工程实现
- 第 7、8 题开始从 RAG 延伸到 Agent 设计
