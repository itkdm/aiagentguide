# LLM 面试题

这部分主要考察你是否理解大模型的基本工作方式，以及能否把模型能力、限制和应用落地讲清楚。

## 建议重点准备

- token、上下文窗口和注意力机制的关系
- 预训练、指令微调、对齐分别解决什么问题
- temperature、top_p、max_tokens 等参数的作用
- 幻觉为什么会出现，常见缓解方法有哪些
- 做 LLM 应用时如何平衡效果、延迟和成本

## 学习建议

- 先把基础概念讲准确，再补充实际项目经验
- 回答参数和策略问题时，尽量说出适用场景和取舍
- 只背定义不够，最好能结合一个线上或实验项目来说明
