# 入门

这一栏面向刚接触 AI Agent、但已经具备基本编程基础的读者。重点不是堆概念，而是先建立一个清晰判断：什么问题值得用 Agent 解决，什么问题其实只需要普通工作流。

## 你会在这里看到什么

- AI Agent 的基本概念
- Agent 和聊天机器人的区别
- 学习顺序与入门路线
- 进入框架和实战前需要建立的判断标准

## 这一栏适合什么时候看

- 第一次系统接触 AI Agent
- 已经看过很多介绍，但还分不清核心概念
- 想在开始选框架之前先把基本认知建立好

## 阅读建议

建议先看完这一栏，再进入 [框架](../frameworks/index.md) 和 [实战](../tutorials/index.md)。如果基础判断不稳，后面很容易陷入“工具看了很多，但不知道该怎么选”的状态。
