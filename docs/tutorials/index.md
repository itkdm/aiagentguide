# 实战教程

这一栏的重点是把 Agent 从概念变成可运行的东西。真正的实战并不是一上来就做复杂系统，而是先跑通一个最小闭环，再逐步扩到更完整的工作流。

## 你会在这里看到什么

- 第一个 Agent 应该怎么下手
- 最小可用实战案例
- 工具调用、状态管理和执行闭环的入门方式
- 从简单任务过渡到更复杂场景的实践顺序

## 这一栏适合什么时候看

- 已经完成基础概念学习
- 已经开始比较框架，准备动手
- 想从一个实际例子理解 Agent 运行方式

## 阅读建议

实战部分建议从单一任务开始，不追求复杂，先保证“能跑通、能解释、能复现”。这比一次性做一个功能很多但不稳定的 Agent 更有价值。
