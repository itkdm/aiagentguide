## DeepEval是什么？

DeepEval 是面向 LLM 和 Agent 应用的自动化测试框架，重点在于把质量评估做成可重复执行的工程环节。

## 主要功能

- 自动化评测
- 生成质量基准
- Agent 行为测试
- 检索结果验证

## 如何使用？

1. 先定义需要评测的目标
2. 为输出质量和行为建立测试样例
3. 反复运行并比较不同版本结果

## 订阅套餐

开源能力通常可以直接使用，平台或商业服务以官网最新说明为准。

## 适用人群

- 需要稳定质量控制的团队
- 想做测试化评测的人
- 不满足于手工对比输出的开发者
