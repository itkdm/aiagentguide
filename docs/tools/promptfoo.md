## Promptfoo是什么？

Promptfoo 是一个更偏工程化的评测工具，适合系统比较不同提示词、模型配置和工作流效果。

## 主要功能

- Prompt 评测
- RAG 测试
- Agent 行为比较
- 红队测试与回归验证

## 如何使用？

1. 定义测试样例和比较目标
2. 跑不同提示词或配置版本
3. 根据结果持续做回归和优化

## 订阅套餐

本地和开源能力通常可以直接使用，若涉及商业能力或团队方案，请以官网当前说明为准。

## 适用人群

- 需要系统评测 LLM/Agent 的团队
- 想把质量测试纳入开发流程的人
- 不希望每次调整都靠主观感受判断的人
