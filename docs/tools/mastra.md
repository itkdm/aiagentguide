## Mastra是什么？

Mastra 是更偏向 TypeScript 和现代 Web 工程体系的 Agent 开发框架，适合直接进入现有前后端项目。

## 主要功能

- TypeScript 友好的 Agent 开发
- 工作流与工具集成
- 更容易进入 Web 应用工程体系

## 如何使用？

1. 在现有 TS/Node 项目里接入框架
2. 定义 Agent、工具和流程
3. 和现有接口、前端或服务端逻辑整合

## 订阅套餐

框架本身可用，实际成本主要取决于模型、部署和你接入的基础设施。

## 适用人群

- TypeScript 团队
- Web 应用开发者
- 想把 Agent 直接放进现有工程的人
