## AutoGen是什么？

AutoGen 是微软推动的多 Agent 框架，更强调代理之间如何通信、协作、分工和形成网络式任务系统。

## 主要功能

- 多 Agent 协作
- 消息驱动交互
- 角色分工
- 异步与事件式执行

## 如何使用？

1. 先定义不同代理的角色和职责
2. 设计代理之间的交互规则
3. 接入模型和工具，让任务在多个代理之间推进

## 订阅套餐

框架本身可开源使用，实际成本主要由模型调用和运行环境决定。

## 适用人群

- 想研究多 Agent 架构的人
- 需要角色协作与代理通信的开发者
- 更关心协作网络而不是单流程图的人
