## Playwright是什么？

Playwright 是成熟的浏览器自动化工具，本身不是 Agent 框架，但很适合作为浏览器型 Agent 的执行底座。

## 主要功能

- 页面控制
- 自动化测试
- 浏览器交互执行
- 稳定复现网页操作流程

## 如何使用？

1. 先把需要的页面流程自动化
2. 把自动化能力封装成可调用工具
3. 再让 Agent 在更高层决定何时调用

## 订阅套餐

项目本身可免费使用，实际成本主要来自执行环境和你接入的基础设施。

## 适用人群

- 需要稳定网页控制的开发者
- 做浏览器自动化和测试的人
- 想给 Agent 提供执行层能力的团队
