## Flowise是什么？

Flowise 是面向 LLM、RAG 和 Agent 流程的可视化搭建工具，适合快速做演示和验证。

## 主要功能

- 可视化流程搭建
- Agent 与 RAG 原型验证
- 节点式连接模型和工具

## 如何使用？

1. 先用节点搭出最小流程
2. 接入模型、文档或工具
3. 在可视化界面里反复测试和调整

## 订阅套餐

通常支持自托管使用，若涉及托管服务或商业能力，请以官网当前说明为准。

## 适用人群

- 想快速做原型的人
- 教学和演示场景开发者
- 更偏流程搭建而不是底层工程的人
