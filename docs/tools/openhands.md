## OpenHands是什么？

OpenHands 是更接近代码任务执行平台的 Agent 工具，重点在于让智能体直接面向开发任务形成闭环。

## 主要功能

- 代码修改
- 命令执行
- 开发任务推进
- 面向工程任务的 Agent 交互

## 如何使用？

1. 给出清晰的软件开发任务
2. 让系统执行修改、运行和验证步骤
3. 根据结果继续修正或推进任务

## 订阅套餐

开源能力通常可以本地使用，若涉及托管或商业化服务，应以官网当前说明为准。

## 适用人群

- 关注 code agent 的开发者
- 软件工程自动化场景
- 想研究开发任务闭环的人
