## Langfuse是什么？

Langfuse 是面向 LLM 与 Agent 系统的观测平台，重点是让你看到调用链、成本、评分和行为轨迹。

## 主要功能

- trace 追踪
- prompt 管理
- 成本与指标观测
- 线上行为分析

## 如何使用？

1. 在应用里接入追踪
2. 记录模型调用和工具链路
3. 基于数据分析表现并继续优化

## 订阅套餐

通常有开源核心和云服务模式，具体订阅套餐以官网当前说明为准。

## 适用人群

- 已经进入迭代阶段的 Agent 团队
- 需要定位失败原因的人
- 想把 LLM / Agent 运行过程可视化的人
