## LangGraph是什么？

LangGraph 是面向复杂 Agent 系统的状态化编排框架，重点不在“把 Agent 做出来”，而在“把 Agent 做稳、做可控”。

## 主要功能

- 状态管理
- 节点式流程控制
- checkpoint 与恢复
- human-in-the-loop

## 如何使用？

1. 先定义流程节点和状态结构
2. 用图结构组织执行顺序与分支
3. 接入模型、工具和记忆机制

## 订阅套餐

框架本身可以开源使用，实际成本主要来自模型调用、部署和配套观测资源。

## 适用人群

- 想做复杂可控 Agent 的开发者
- 需要状态恢复和人工介入的团队
- 希望把 Agent 系统往生产级推进的人
