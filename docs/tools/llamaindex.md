## LlamaIndex是什么？

LlamaIndex 更偏数据接入、索引和检索增强层，适合把文档和知识体系接进 LLM 或 Agent 系统。

## 主要功能

- 数据接入
- 文档索引
- 检索增强生成
- 与 Agent 结合的知识层支持

## 如何使用？

1. 先接入文档或数据源
2. 建立索引和检索逻辑
3. 把检索结果接入聊天或 Agent 流程

## 订阅套餐

开源能力可以使用，若涉及云服务或企业级能力，请以官网当前说明为准。

## 适用人群

- 做知识库与文档问答的人
- 需要检索增强能力的团队
- 想把数据层打牢后再做 Agent 的开发者
