## AstronClaw 是什么？

<div class="tool-prose-card">
  <p>AstronClaw 是讯飞星辰 Agent 平台上的一项云端 AI 助手服务。官方文档说明它基于 OpenClaw 核心能力，面向不想本地折腾环境、配置密钥或长期保持设备开机的用户，提供浏览器访问、云端部署、技能市场和消息渠道接入等能力。</p>
  <p>和需要本地 CLI、Docker 或持续开机的方案相比，AstronClaw 更强调“全天候在线服务”的产品形态。你可以把它理解为一个由平台托管的个人 AI 助手，适合直接在云端接任务、扩展技能并接入企业协作渠道。</p>
</div>

## 主要功能

<div class="tool-feature-grid">
  <article class="tool-feature-card">
    <h3>云端一键部署</h3>
    <p>官方文档说明可在网页端直接“立即部署”，大约 1 分钟完成，无需准备本地 CLI 或 Docker 环境。</p>
  </article>
  <article class="tool-feature-card">
    <h3>7×24 在线运行</h3>
    <p>与本地 OpenClaw 相比，AstronClaw 由平台托管，电脑关闭后也不会中断运行，更适合作为持续在线的云端助手。</p>
  </article>
  <article class="tool-feature-card">
    <h3>技能市场与自定义技能</h3>
    <p>支持浏览和安装官方技能，也支持上传 zip 文件创建自定义技能。当前官方文档写明已提供 131 个官方技能。</p>
  </article>
  <article class="tool-feature-card">
    <h3>多渠道接入</h3>
    <p>支持飞书、钉钉、企业微信等企业协作平台接入，方便在 IM 渠道里派发任务、查看进度并接收结果。</p>
  </article>
</div>

## 如何使用？

<div class="tool-step-list">
  <article class="tool-step-card">
    <span class="tool-step-no">01</span>
    <h3>打开 AstronClaw 页面并部署</h3>
    <p>进入讯飞星辰 Agent 平台中的 AstronClaw 页面，点击“立即部署”，等待云端初始化完成。</p>
  </article>
  <article class="tool-step-card">
    <span class="tool-step-no">02</span>
    <h3>进入对话页开始交互</h3>
    <p>部署完成后进入对话页面与 AstronClaw 交互。官方文档说明支持上传 pdf、doc、txt、jpg 等文件，并给出了单文件和总量限制。</p>
  </article>
  <article class="tool-step-card">
    <span class="tool-step-no">03</span>
    <h3>安装官方技能或上传自定义技能</h3>
    <p>进入 Skills 市场安装需要的能力，或上传 zip 文件创建自定义技能，以扩展不同任务场景。</p>
  </article>
  <article class="tool-step-card">
    <span class="tool-step-no">04</span>
    <h3>接入企业协作渠道</h3>
    <p>在渠道配置页面接入飞书、钉钉或企业微信，让 AstronClaw 在企业协作工具中接收任务和返回结果。</p>
  </article>
</div>
