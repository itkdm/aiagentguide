## browser-use是什么？

browser-use 是把网页环境直接暴露给 Agent 操作的一类工具，适合做浏览器型 Agent。

## 主要功能

- 网页操作
- 表单填写
- 页面信息采集
- 登录和浏览器流程执行

## 如何使用？

1. 确定需要在哪些网页上执行任务
2. 把页面操作能力接入 Agent 流程
3. 通过执行结果不断调整步骤和容错逻辑

## 订阅套餐

项目本身通常可用，实际成本主要来自模型调用和浏览器执行环境。

## 适用人群

- 想做 browser agent 的开发者
- 网页自动化任务场景
- 需要从页面采集和执行操作的人
