# 项目

这一栏用于整理 AI Agent 相关的开源项目，但重点不是“越多越好”，而是帮助读者快速理解不同项目代表的方向、能力边界与学习价值。

## 页面演示

下面这两个示例就是新接入的公共组件，进入项目概览页就能直接看到实际渲染效果。

### 项目阶段流程图

## 你会在这里看到什么

- 不同方向的开源项目整理
- 按语言和能力路线区分的项目分类
- 适合入门、学习、参考和二次开发的项目入口
- 帮助快速判断一个项目值不值得花时间研究的基础信息

## 这一栏适合什么时候看

- 想找学习样例
- 想找能参考架构的开源仓库
- 想了解当前不同 Agent 路线分别有哪些代表项目

## 阅读建议

项目栏目更适合在“已经知道自己想做哪类 Agent”之后再看。这样你不会陷入收藏了很多仓库，但最后一个都没有认真读完的状态。
