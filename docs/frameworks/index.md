# 框架

这一栏既关注“怎么选框架”，也关注“框架具体怎么用”。不同框架背后的设计重点并不一样，有的强调状态和流程控制，有的强调多 Agent 协作，有的更偏平台化和可视化交付；选型之外，我们也会继续补齐每个框架的入门、核心能力、进阶用法和实战文档。

## 已收录框架

- [LangChain](/frameworks/langchain/)

## 你会在这里看到什么

- 主流 AI Agent 框架的定位差异
- 适用场景与学习门槛
- 代码驱动和平台驱动路线的区别
- 框架选型时最容易忽视的判断点
- 各个框架的基础使用文档与能力拆解
- 围绕具体框架的进阶能力、最佳实践和实战示例

## 这一栏适合什么时候看

- 已经理解 Agent 基本概念，准备开始做技术选型
- 想知道不同框架到底差别在哪里
- 已经确定了某个框架，想系统查看它的使用文档
- 想避免上来就盲目跟风某一个热门框架

## 阅读建议

这一栏更适合带着目标来看。你可以先从框架概览页判断选型方向，再进入具体框架，例如 LangChain，继续看它的入门、核心组件和高级能力文档；这样既能先做判断，再深入落地，也方便后续扩展到 AutoGen、LangGraph 等其他框架。
