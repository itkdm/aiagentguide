# LangSmith Studio

LangSmith Studio 是用于本地与已部署 Agent 的可视化开发和调试入口。

它通常用于：

- 调试本地 Agent Server
- 连接已部署 Agent
- 查看线程与中断
- 快速验证 Agent 行为

> [!TIP]
> 如果你想先在本地运行 Agent，再连接 Studio，可以参考本地 Agent Server 的相关文档。部署后的 Agent 同样可以接入 Studio。
