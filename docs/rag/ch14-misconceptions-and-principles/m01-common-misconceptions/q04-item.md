# 14.1.4 相似度高不等于答案就一定对，为什么？

先给结论：*相似度只说明“语义接近”，不说明“证据足够”。答案正确与否取决于证据是否完整、是否支持结论。*

## 相似度只是“相关性”，不是“正确性”

向量相似度衡量的是“语义接近”，但答案正确还需要：

- 关键条件是否齐全  
- 时效性是否匹配  
- 证据是否一致  

这就是为什么需要评估与证据检查机制，而不能只看相似度分数。citeturn1search0

## 典型失效场景

- 命中了类似问题，但条件不同  
- 命中了旧版本知识  
- 命中了片段答案，缺少限制条件  

这些问题在“相似度高”的情况下依然会发生。

## 该如何避免

最有效的办法是把“证据充分性”纳入评估：

- 建立最小评测集  
- 对比不同版本的答案  
- 检查证据覆盖率  

评估不是奢侈品，而是让“相似度”不再误导你的基本手段。citeturn1search0

## 一句话总结

相似度高只能说明“看起来像”，不能保证“答案正确”。正确性必须由证据完整性与评估来保障。
