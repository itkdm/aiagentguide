# 第 5 章 指令微调与对齐

先给结论：**预训练让模型“会说话”，但不保证它“会做助手”。要让模型稳定执行指令、偏好一致且可控，必须做指令微调与对齐。** 

这一章解决的是“从语言能力到助手能力”的工程鸿沟。你会看到：为什么纯预训练模型看似聪明却不稳定、为什么指令遵循需要单独训练、以及对齐方法如何把模型行为收敛到可用范围。

## 你应该建立的判断

- 预训练目标是“预测下一个 token”，而不是“遵循指令”
- 指令遵循是行为策略，需要额外的监督或偏好信号
- 对齐不是一招解决，而是训练、数据与系统约束的组合
