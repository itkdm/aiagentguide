# 5.3 对齐（Alignment）

先给结论：**对齐是让模型行为更贴近“人类意图与可接受规范”的过程，它不是提升能力，而是把能力约束到可用、可控与安全的范围。** 

这一节不会停留在概念解释，而是让你理解：对齐到底在控制什么、为什么能力强不代表能用，以及对齐如何直接影响产品体验与上线安全。

## 这一节会回答什么问题

- [什么是对齐（Alignment）？](./q01-what-is-alignment)
- [为什么大模型不仅要“能力强”，还要“行为可控”？](./q02-why-behavior-control-matters)
- [对齐为什么会影响模型风格、安全性和可用性？](./q03-alignment-affects-ux)

## 学完后你应该建立的判断

- 对齐解决的是“行为策略”，不是“语言能力”
- 没有对齐的模型在真实系统里很难稳定交付
- 对齐程度会直接改变产品体验与风险边界
