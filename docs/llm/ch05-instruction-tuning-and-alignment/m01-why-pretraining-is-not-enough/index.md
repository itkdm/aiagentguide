# 5.1 为什么预训练后还不够

先给结论：**预训练的目标是“像语料一样继续说”，而不是“按用户意图完成任务”。所以即使模型有语言能力，也不稳定、不可控，还缺少助手行为。** citeturn0academia13turn0search4

这部分会把“会补全文本”和“会理解并执行意图”拆开解释，让你明确哪些问题必须靠指令微调和对齐来解决。

## 学完后你应该形成的判断

- 预训练目标决定了模型的“续写性”，而不是“执行性”
- 指令遵循是行为对齐问题，不是语言能力问题
- 对齐需要训练数据、偏好信号与系统约束共同完成
