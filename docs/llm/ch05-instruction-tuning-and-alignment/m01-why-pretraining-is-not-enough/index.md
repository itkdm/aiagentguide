# 5.1 为什么预训练后还不够

先给结论：**预训练的目标是“像语料一样继续说”，而不是“按用户意图完成任务”。所以即使模型有语言能力，也不稳定、不可控，还缺少助手行为。** 

这部分要解决的是一个很容易被忽视的问题：为什么一个“已经很聪明”的预训练模型，拿到真实产品里却仍然经常不好用。

很多人第一次接触 LLM 时都会有一个直觉误区：既然模型已经能写文章、做问答、甚至写代码，那离“助手”应该只差一层 Prompt。  
但真实情况并不是这样。预训练主要学到的是语言模式和统计规律，而不是“遇到用户请求时应该采取什么行为策略”。

这一节会把“会补全文本”和“会理解并执行意图”拆开，帮助你建立一个关键判断：**语言能力和助手能力不是一回事，后者必须靠额外训练和行为约束来塑形。**

## 学完后你应该形成的判断

- 预训练目标决定了模型的“续写性”，而不是“执行性”
- 指令遵循是行为对齐问题，不是语言能力问题
- 对齐需要训练数据、偏好信号与系统约束共同完成
