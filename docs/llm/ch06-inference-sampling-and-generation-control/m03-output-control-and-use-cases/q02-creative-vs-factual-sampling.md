# 6.3.2 为什么创意生成和事实问答适合不同的采样参数？

先给结论：**创意生成需要多样性，事实问答需要稳定性与可验证性，因此采样参数应该完全不同。** citeturn2view0turn0search2turn3search4

## 创意任务：更高温度、更宽候选

创意任务的目标是“多样与新颖”，可容忍一定不确定性：  
因此更适合使用较高 temperature 或更高 top-p/top-k。citeturn2view0turn1search0

## 事实任务：更低温度、更窄候选

事实问答更关注“可验证与稳定”，错误代价高：  
因此应使用较低 temperature、更保守的 top-p/top-k 或 greedy。citeturn0search2turn3search4

## 一个可执行的经验策略

你可以先用这样的策略起步：

- 事实类：低温度 + 低 top-p
- 创意类：中高温度 + 中高 top-p
- 需要稳定格式：偏向 greedy 或低温度

然后再根据回归测试结果微调参数。citeturn0search2turn1search0

## 常见误区

- 事实任务使用高温度，导致不稳定与幻觉增加
- 创意任务使用低温度，导致输出单一、缺乏灵感

你可以先记住一句话：**采样参数的选择，首先取决于错误代价。**
