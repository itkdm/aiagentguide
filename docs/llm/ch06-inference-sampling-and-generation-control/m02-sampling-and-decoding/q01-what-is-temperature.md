# 6.2.1 什么是 temperature？

先给结论：**temperature 用来调节采样分布的“平滑程度”，值越高随机性越强，值越低越接近确定性。** 

## temperature 做了什么

在采样前，模型会把 logits 转成概率分布。  
temperature 的作用就是“放大或压缩分布差异”，从而改变抽样的随机性。

直观理解：

- **低温（接近 0）**：更接近贪心解码，回答更稳定
- **高温**：分布更平滑，回答更发散

## 工程上如何用

可以按任务类型决定温度：

- **事实类、规则类**：温度偏低，保证稳定性
- **创意类、发散类**：温度适当提高，扩大多样性

温度不是越低越好，过低会导致“僵化重复”。

## 常见误区

- 以为 temperature 能提升模型能力  
  实际上它只改变输出分布，不改变模型本身。

## 你可以先记住一句话

**temperature 决定“随机性强弱”，不决定“回答质量上限”。**
