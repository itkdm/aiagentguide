# 6.2.5 为什么同样的问题回答会不同？

先给结论：**回答不同主要来自采样随机性、解码策略差异，以及上下文的微小变化。** 

## 1. 采样是随机过程

只要 temperature、top-k 或 top-p 不是“极端确定”，就会有随机性。  
这意味着同样的输入也可能采样到不同的 token 序列。

## 2. 解码策略决定输出分布

greedy 解码只取最大概率 token，输出最稳定；  
top-k/top-p 在候选集合里采样，会产生多样化答案。

## 3. 上下文变化会放大差异

哪怕提示里多一个标点或一个换行，也可能改变概率分布，最终让输出分岔。  
这就是为什么提示工程强调“格式稳定”。

## 工程上的对策

你可以用这些方式降低不确定性：

- 降低 temperature 或收紧 top-p/top-k
- 固定系统提示与格式
- 对关键任务做多次采样并投票

## 结论收束

**输出差异不是“模型不稳定”，而是“采样本来就有随机性”。**
