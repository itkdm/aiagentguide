# 6.2 采样与解码策略

先给结论：**解码策略决定“从概率分布里选哪个 token”，温度控制随机性，top-k/top-p 控制候选范围。** 

这一节要解决的是另一个非常常见的误解：  
很多人以为模型回答不一样，是因为“模型在瞎发挥”或者“模型不稳定”；但实际上，只要解码策略允许随机性，同一个概率分布就完全可能导出不同结果。

所以这部分的重点不只是解释几个参数名，而是帮你建立一套更稳的理解：

- 温度到底在改变什么
- top-k 和 top-p 为什么都在“截断候选”，但思路不同
- greedy decoding 为什么更稳定，却不一定更好
- 为什么采样差异会表现成风格差异、创造性差异和稳定性差异

理解这些之后，你就不会再把所有输出波动都误判成“模型能力问题”。

## 学完后你应该建立的判断

- 采样参数影响的是“输出分布”，不是“模型能力”
- 越稳定的解码方式，越容易失去多样性
- 采样策略应该跟任务性质匹配
