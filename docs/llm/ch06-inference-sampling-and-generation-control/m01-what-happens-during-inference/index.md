# 6.1 模型推理时到底在做什么

先给结论：**推理阶段就是“用当前上下文计算下一 token 分布，并一步步解码成文本”。成本主要来自注意力计算与多次解码循环。** 

这一节会把推理过程拆成可理解的步骤，并说明为什么“没有训练参数更新”仍然非常耗资源。

## 这一节会回答什么问题

- [LLM 推理时到底在做什么？](./q01-what-happens-during-inference)
- [从输入到输出，中间发生了哪些关键步骤？](./q02-key-steps-input-to-output)
- [为什么推理阶段虽然不训练参数，但仍然很耗资源？](./q03-why-inference-is-expensive)

## 学完后你应该建立的判断

- 推理的核心是“逐 token 预测 + 解码”
- 注意力计算与上下文长度决定了推理成本
- 采样策略决定“回答风格”，而非“模型能力”
