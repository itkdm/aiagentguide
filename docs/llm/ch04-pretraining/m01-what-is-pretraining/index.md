# 4.1 什么是预训练

这部分会先解释预训练到底是什么，以及为什么它是大语言模型能力形成的基础。

预训练并不是某个可选技巧，而是 LLM 能力的起点：模型在大规模通用语料上先学到通用语言表示，再把这些能力迁移到具体任务上。

## 学完这一节后，你应该建立的几个判断

- 预训练的核心是“通用表示学习”，不是某个单一任务的优化
- LLM 的很多能力不是靠后续微调“加上去的”，而是预训练阶段“学出来的”
- 预训练和传统监督学习的差异，根源在目标、数据规模和任务迁移方式
