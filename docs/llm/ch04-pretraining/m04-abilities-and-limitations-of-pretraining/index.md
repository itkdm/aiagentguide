# 4.4 预训练带来的能力与局限

这部分会回到结果层面，解释预训练为什么能让模型具备语言能力，同时也说明它为什么仍然有“只会续写、不一定会当助手”的局限。

这一节的重点是把三件事拆开：

- 预训练如何带来通用语言能力
- 为什么预训练模型“会续写”，但不一定“会做助手”
- 为什么预训练知识会过时，以及工程上如何处理

## 这一节会回答什么问题

- [预训练为什么能带来语言理解与生成能力？](./q01-why-pretraining-brings-language-ability)
- [为什么预训练模型“会续写”，但不一定“会做助手”？](./q02-can-complete-but-not-assistant)
- [为什么预训练阶段学到的知识会过时？](./q03-why-pretraining-knowledge-gets-stale)

## 学完这一节后，你应该建立的几个判断

- 预训练提供的是通用语言能力，但不自动保证对齐或可靠性
- “会续写”与“会当助手”之间差的是指令对齐与行为约束
- 预训练知识是静态快照，更新需要额外机制（微调、检索、工具）
