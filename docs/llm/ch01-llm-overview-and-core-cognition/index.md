# 第 1 章 LLM 全景图与核心认知

先给结论：**学习 LLM 的第一步，不是先钻进细节，而是先把整体图景搭起来。你需要先知道 LLM 在 AI 体系里处于什么位置、它的核心能力从哪里来、它和传统 NLP 或规则系统到底有什么不同。**

这一章的目标，就是把大语言模型的基本认知先立住。很多人一上来就看 Transformer、Attention、RAG 或 Agent，很容易学到后面概念越来越多，但整体判断越来越乱。原因通常不是知识不够，而是没有先把“全景图”建立起来。

这一章会先回答最基本、也最容易混的问题：LLM 是什么，不是什么；它为什么看起来像“什么都能做”，但又经常出错；它适合解决哪些问题，又不适合解决哪些问题。

## 你应该建立的判断

- LLM 是语言模型和基础模型能力的结合体，不只是“更大的聊天工具”
- 它的强项主要在语言理解、生成和模式泛化，不等于天然可靠的知识库或执行系统
- 后续学 Transformer、预训练、对齐、RAG 和 Agent 时，都应该回到这一章建立的整体视角来理解
