# 1.1 LLM 的定义与定位

这部分会先帮你建立对大语言模型的基本认识，回答它是什么、和传统 NLP 模型有什么不同，以及它和生成式 AI、AIGC、基础模型之间到底是什么关系。

这一节最重要的任务，不是先记住一堆名词，而是先把几个经常混在一起的概念拆开：

- LLM 首先是一类语言模型，不是一个具体产品
- 它和传统 NLP 的差异，不只是“参数更多”
- 它表现出通用能力，和训练方式、数据规模、模型架构都有关系
- 生成式 AI、AIGC、基础模型、LLM 之间是包含关系和交叉关系，不是同义词

如果这些边界一开始没有建立起来，后面再看 Transformer、预训练、对齐、Prompt、RAG 和 Agent 时，就很容易一直带着模糊理解往前学。

## 学完这一节后，你应该建立的几个判断

- 当你说“LLM”时，指的是一种以大规模预训练为基础、能建模和生成语言的模型范式，而不只是聊天机器人
- 当你比较 LLM 和传统 NLP 时，应该先比较“范式”，再比较“指标”
- 当你看到模型能做翻译、总结、问答、代码生成时，应该理解这是统一语言建模能力在不同任务上的展开，而不是很多孤立能力的简单拼装
- 当你看到 Foundation Model、生成式 AI、AIGC 这些词时，应该能分清谁是更上层的概念，谁是更具体的模型类型
