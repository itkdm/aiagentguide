# 7.2 模型的“记忆”到底是什么

先给结论：**大模型的“记忆”主要有两类：参数记忆（知识内化在参数里）与上下文记忆（提示中临时可见的内容）。两者性质完全不同。** citeturn0search1turn0search6turn0search9

这一节会帮你区分“模型看起来记得”和“模型真的能稳定调用”的差异，避免把上下文当成长期记忆、或把参数当成稳定知识库。citeturn0search1turn0search9

## 学完后你应该建立的判断

- 参数记忆是“长期但不一定可控”的知识来源
- 上下文记忆是“临时但可控”的任务信息来源
- 稳定调用需要提示组织或检索机制支持
