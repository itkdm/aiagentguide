# 7.3.4 长文本输入会带来哪些典型问题？

先给结论：**长文本场景最常见的问题是“关键信息被忽略、指令被稀释、输出偏离目标”，其根源往往是位置偏好与信息过载。** 

## 典型问题清单

1. **中间信息失效**  
   关键信息放在中间时，模型容易忽略。

2. **指令被背景稀释**  
   大量背景内容覆盖了任务指令，模型输出偏离。

3. **答案不稳定**  
   信息越多，模型的回答路径越不确定，结果更不稳定。（基于研究结论的推断）

## 一组快速自查方法

你可以用下面三种方式快速定位问题：

- **位置敏感测试**：把关键事实放到开头、中间、结尾分别测试  
- **长度增量测试**：逐步增加上下文长度，观察性能变化  
- **噪声插入测试**：添加无关信息，检查是否被干扰  

## 结论收束

**长文本问题不是“随机故障”，而是结构性偏差，需要结构化应对。**
