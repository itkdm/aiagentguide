# 7.1.1 什么是上下文窗口（Context Window）？

先给结论：**上下文窗口就是模型在一次推理中可处理的最大序列长度，通常由模型配置中的最大位置长度决定。** citeturn2search3

## 工程化理解

你可以把上下文窗口理解成“模型一次能看到多少 token”。  
超过这个长度的内容不会进入当前推理，因此不会影响输出。citeturn2search3

在很多模型的配置中，这个上限直接由 `max_position_embeddings` 等参数约束。citeturn2search3

## 一个常见误区

上下文窗口不是“记忆容量”，而是“当前可见范围”。  
模型不会自动记住窗口之外的内容，除非你把它重新放进上下文里。citeturn2search3

## 你可以先记住一句话

**上下文窗口决定模型“能看到什么”，而不是“能记住什么”。**
