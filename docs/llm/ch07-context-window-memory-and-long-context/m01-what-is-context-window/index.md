# 7.1 上下文窗口是什么

先给结论：**上下文窗口就是模型一次推理能“看到”的最大序列长度，它是模型结构与配置决定的硬限制。** 

这一节会把上下文窗口讲清楚，帮助你理解它为什么会限制模型一次能处理的信息量，以及为什么这会成为产品设计中的关键约束。

- [上下文窗口为什么会限制模型一次能处理的信息量？](./q02-why-context-window-limits-information)
- [为什么上下文长度是 LLM 产品设计中的关键约束？](./q03-why-context-window-is-key-constraint)

## 学完后你应该建立的判断

- 上下文窗口是“硬上限”，不是软建议
- 超出窗口的信息不会参与当前推理
- 长上下文带来成本与质量风险，需要工程控制
