# 3.3.4 Attention、MLP、残差连接、LayerNorm 分别起什么作用？

先给结论：

**Attention 负责上下文信息的动态聚合，MLP 负责非线性变换和特征提升，残差连接保证信息与梯度流动，LayerNorm 让训练更稳定。**

这四个组件合在一起，才构成了 Transformer block 的核心能力。

## Attention：上下文动态聚合

Attention 的作用可以理解成：

- 根据相关性，从序列中“取回”有用信息
- 更新当前 token 的上下文化表示

它解决的问题是：

- 如何让远处信息直接参与当前表示
- 如何让模型按需聚焦关键上下文

所以 Attention 是 Transformer 建模“关系”的核心部件。

## MLP / FFN：非线性特征变换

Attention 负责“从哪里拿信息”，但还需要一个模块负责“怎么加工信息”。

这就是 MLP（或称 FFN）。

它的作用是：

- 对每个位置的表示做非线性变换
- 提升表示的表达能力
- 让模型能形成更复杂的组合模式

可以把它理解成：

- Attention 决定“看什么”
- MLP 决定“看完后怎么加工”

两者协同，才能形成强表达力。

## 残差连接：信息保留与梯度通道

深层网络最常见的问题之一是：

- 信息在多层变换中逐渐被冲淡
- 训练时梯度难以有效传回

残差连接的作用就是：

- 让原始输入能直接绕过子层，保留关键信息
- 提供更稳定的梯度通道，缓解深层训练困难

所以残差不是“锦上添花”，而是深层模型可训练性的基础。

## LayerNorm：稳定数值分布

深层模型训练时，数值分布很容易漂移。

LayerNorm 的作用是：

- 在每个位置上做归一化
- 稳定激活分布
- 减少训练不稳定

它不会替代 Attention 或 MLP，但会显著提升训练稳定性和收敛表现。

## 这些组件如何协同工作

一个典型 block 可以理解成这样的流程：

1. 用 Attention 聚合上下文
2. 加残差 + LayerNorm，保持信息流稳定
3. 用 MLP 做非线性加工
4. 再加残差 + LayerNorm

这样结构既能不断引入新关系信息，又能避免深层堆叠带来的训练崩溃。

## 一个常见误区

有些人会以为 Transformer 的“聪明”只来自 Attention。

这是不完整的理解。真正的能力来自：

- Attention 提供关系建模
- MLP 提供特征变换
- 残差和 LayerNorm 提供稳定训练和信息保留

缺一不可。

## 你可以先这样记住

- Attention 解决“看哪里、怎么聚合”
- MLP 解决“怎么加工和提升表示”
- 残差连接保证信息和梯度流动
- LayerNorm 让训练更稳定
- Transformer 的能力来自这些组件协同，而不是单点魔法
