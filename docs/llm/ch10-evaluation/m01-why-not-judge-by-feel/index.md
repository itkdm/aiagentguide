# 10.1 为什么不能只靠主观感觉评模型

先给结论：**“感觉这个模型挺聪明”只能作为最初印象，不能作为评测结论。因为 LLM 很容易在少量样例里表现惊艳，却在规模化、长尾样本和真实流程里暴露问题。**

这部分先帮你建立一个基本防线：不要把主观感受误当成评测结果。你会看到，为什么“会聊天”不等于“可上线”，以及为什么离线能力和线上效果必须拆开看。
