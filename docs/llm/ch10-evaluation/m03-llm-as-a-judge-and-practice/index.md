# 10.3 LLM-as-a-Judge 与评测实践

先给结论：**LLM-as-a-Judge 很有用，但它不是“自动评测万能解”。它适合扩大评测规模、覆盖开放式输出，却依然会受偏差、提示设计和评分标准影响，所以必须和客观指标、人工评审一起用。**

这部分会带你把评测方法放到真实工程里看。重点不是某个术语本身，而是你该怎么组合这些方法，才能既有规模，又不丢掉可信度。
