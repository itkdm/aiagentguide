# 10.4 线上模型评估

先给结论：**LLM 系统上线后，真正决定成败的不是某次离线分数，而是线上质量、稳定性、延迟、成本和错误恢复能力能不能长期被看见、被定位、被修正。**

这部分回到生产视角，讲的是“系统已经上线之后，你该怎么看它是不是在变差、为什么变差、以及怎么尽快定位问题”。

- [如何判断模型质量下降是模型问题、Prompt 问题还是数据问题？](./q02-how-to-locate-quality-drop)

## 读完后你应该能判断

- 哪些线上指标是真正值得长期追踪的
- 一次质量下降更可能出在模型、prompt、检索、工具还是数据分布
- 为什么没有线上监控的 LLM 产品很难稳定迭代
