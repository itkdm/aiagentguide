# 2.3 语言模型与自回归生成

这部分会从语言模型和自回归生成出发，帮助你理解 LLM 为什么总是按顺序逐 token 地生成文本，以及这种过程为什么会让模型看起来像在“思考”。

这一节的关键，是把三个经常被混在一起的概念拆开：

- 什么是语言模型
- 什么是自回归生成
- 为什么这种逐步生成机制会让输出呈现出过程感和推理感

如果这三件事没分清，后面你很容易把：

- 训练目标
- 推理过程
- 表面行为现象

混成一个东西来看。

## 学完这一节后，你应该建立的几个判断

- 语言模型的本质，是对 token 序列概率分布进行建模
- 自回归生成不是产品层技巧，而是许多 LLM 的基础生成方式
- “看起来像在思考”更多是逐步生成、上下文反馈和推理文本模式共同形成的效果，不等于模型具有人类式思维过程
