# 8.1 Prompt 的基本作用

先给结论：**Prompt 的本质是“把当前任务、约束和期望输出组织进上下文”，它会显著影响模型行为，但不会凭空给模型增加不存在的知识或工具能力。** 

这一节会先讲清楚 Prompt 是什么、Prompt Engineering 到底在做什么，以及为什么 Prompt 很重要但又不能被神化。

## 这一节会回答什么问题

- [什么是 Prompt？](./q01-what-is-prompt)
- [Prompt Engineering 的本质是什么？](./q02-what-is-prompt-engineering)
- [为什么 Prompt 很重要，但又不能神化 Prompt？](./q03-why-prompt-matters-but-not-magical)

## 学完后你应该建立的判断

- Prompt 是任务组织手段，不是知识注入魔法
- Prompt 好坏主要影响稳定性、可控性和任务对齐
- 真实系统里，Prompt 必须和检索、工具、评测一起看
