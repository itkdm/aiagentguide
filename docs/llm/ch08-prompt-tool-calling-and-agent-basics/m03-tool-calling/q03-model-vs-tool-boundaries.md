# 8.3.3 哪些能力适合交给模型，哪些能力应该交给工具？

先给结论：**模型适合做解释、规划、归纳和语言组织；工具适合做查询、计算、执行和校验。** 

## 更适合交给模型的事情

- 解释复杂概念
- 总结多段材料
- 把结构化数据组织成自然语言
- 基于给定事实做规划与表达

这些任务的共同点是：需要语言压缩和语义组织。

## 更适合交给工具的事情

- 查数据库
- 调外部 API
- 做精确计算
- 执行代码
- 获取最新事实

这些任务的共同点是：需要确定性、实时性或真实执行能力。

## 一个简单的判断标准

你可以问自己三个问题：

1. 这件事有没有唯一可验证结果？  
2. 这件事需不需要实时数据？  
3. 这件事是不是要真的执行外部动作？  

如果答案多半是“是”，那它更适合工具。

## 结论收束

**模型负责“想和说”，工具负责“查和做”。**
